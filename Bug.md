1. # **ID Bug2**

2. # **Заголовок:**
Кнопка "Очистить корзину" не работает

3. # **Описание:**
При ошибке выбора пицц не работает кнопка "Очистить корзину"

3.1 **Предусловия:**
При добавлении две пиццы (Гавайская, Мясная, Маргарита и т.д.) не чистится корзина

3.2 **Тестовые данные:**
Не требуется

3.3 **Шаги воспроизведения:**
1. Зайти на сайт - https://slqamsk.github.io/cases/pizza/v09/cart.html?cart=3%3A2
2. Добавить одну или две пиццы (нап. Мясная) в корзину
3. Перейти в раздел "Корзина"
4. Нажать кнопку "Очистить корзину"

3.4 **Ожидаемый результат:**
После добавления пиццы в корзину при нажатии кнопки "Очистить корзину" очищается корзина

3.5 **Наблюдаемый результат:**
После нажатии кнопки "Очистить корзину" никакой реакции

4. # **Найден в версиях:**
Версия v09 - https://slqamsk.github.io/cases/pizza/v09/index.html?cart=3%3A2

5. # **Критичность:**
Высокая

6. # **Приоритет:**
Высокий
