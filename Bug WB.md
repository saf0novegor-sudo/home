1. # **ID Bug2**

2. # **Заголовок:**
В приложении Wildberries При применении фильтра "Со скидкой" в каталоге отображаются товары без скидок

3. # **Описание:**
Не отображаются товары без скидок в приложении Wildberries когда применяем фильтр "Cо скидкой"

3.1 **Предусловия:**
1. Открыто приложение Wildberries.
2. Выбрана любая категория товаров (напр., "Одежда").
3. В категории более 100 товаров.

3.2 **Тестовые данные:**
Категория: "Кроссовки". Фильтр: "Со скидкой".

3.3 **Шаги воспроизведения:**
1. В разделе "Каталог" открыть категорию "Кроссовки".
2. Нажать "Фильтры".
3. Активировать переключатель "Со скидкой".
4. Нажать "Показать ... товаров".

3.4 **Ожидаемый результат:**
В результатах фильтрации отображаются только товары, у которых есть скидка (цена со strike-through и новая цена)

3.5 **Наблюдаемый результат:**
В результатах, среди товаров со скидкой, периодически встречаются товары по полной цене (без перечеркнутой старой цены).

4. # **Найден в версиях:**
https://www.wildberries.ru/

5. # **Критичность:**
Низкая

6. # **Приоритет:**
Низкий

# **Объяснение выбора классификации:**
Severity: Низкая – Основная функция фильтра работает, но не идеально точно. Пользователь может получить не совсем релевантные результаты, но это не блокирует возможность покупки.

Priority: Низкий – Баг не критичен для совершения заказа и не вызывает финансовых потерь. Может быть исправлен в плановом порядке.
